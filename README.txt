WP Safe Media Plugin is to prevent the used media files from being deleted.

== Description ==

WP Safe Media Plugin is a WordPress plugin for preventing the images which are attached to posts or terms from being deleted. There will be more post type support added in near future. By using this plugin, you can make sure you're not deleting the images which are attached to any posts or terms.

*   You can see the post and terms ids where image is attached to.
*   Image con not be deleted from any delete link of the image.

Few Notes:
* You need to remove the image from posts or terms to delete the image.


== Installation ==

Follow the below steps to install the plugin.

1. Download
2. Upload to your `/wp-contents/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
